# anydb

A collection of high-performance embedded database crates for Rust.

## Crates

- **[`rawdb`](../crates/rawdb/)** - Single-file, low-level and space efficient storage engine with filesystem-like API
- **[`vecdb`](../crates/vecdb/)** - High-performance mutable persistent vectors built on rawdb
- **[`vecdb_derive`](../crates/vecdb_derive/)** - Derive macros for compression support
- **[`vecdb_bench`](../crates/vecdb_bench/)** - Benchmarking suite for vecdb
